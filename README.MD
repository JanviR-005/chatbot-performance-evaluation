# Chatbot Performance Prediction

## Overview
Predicts key performance metrics of chatbot systems using regression models.

## Installation
1. Clone the repo: `git clone https://github.com/JanviR-005/chatbot-performance-evaluation.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Open the Jupyter notebook: `notebook.ipynb`

## Usage
- Run the notebook to train models and generate predictions.
- Predictions are saved in `predictions.csv`.

## Features
- Linear, Polynomial, GLM, and Poisson regression
- Model evaluation and performance comparison
- Prediction for unseen data

## Tech Stack
- Python
- Jupyter Notebook
- NumPy, pandas, scikit-learn, matplotlib
